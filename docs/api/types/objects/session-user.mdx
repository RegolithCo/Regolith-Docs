SessionUser is the type we use to link users who are not the owner to the
session

```graphql
type SessionUser {
  captainId: ID
  createdAt: Timestamp!
  isPilot: Boolean!
  loadout: MiningLoadout
  owner: User
  ownerId: ID!
  sessionId: ID!
  sessionRole: String
  shipName: String
  shipRole: String
  state: SessionUserStateEnum!
  updatedAt: Timestamp!
  vehicleCode: String
}
```
