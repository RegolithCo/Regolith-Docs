# Regolith Survey Corps

  <img src="/img/SurveyorLogo.svg" alt="Surveyor Logo" style={{ float: 'left', marginRight: '1em', width: 120 }} />
The Regolith Survey Corps is a community-driven initiative to gather feedback and insights from users of the Regolith desktop environment. By participating in the survey, you help us understand user needs, improve the user experience, and shape the future of Regolith.
